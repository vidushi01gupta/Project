# Turning E-Waste into Rewards
It is a community-focused web platform that encourages responsible electronic waste disposal by offering reward points to users.
It is designed especially for hostels, colleges, and residential societies, making e-waste recycling simple, trackable, and rewarding.
## Objective
To build a user-friendly e-waste management system where:
* Users can schedule pickups or locate nearby drop-off points for their e-waste.
* Proper disposal is rewarded through a points-based system.
* Communities become more aware and actively participate in e-waste recycling.
## Key Features
### User Authentication & Login
Simple login system for users to manage their e-waste activities.
### Schedule Pickup Requests
Easy-to-use pickup form to request collection from hostels, colleges, or societies.
### Pickup Status Tracking
A dashboard displays status of requests (e.g., pending, collected) and user points.
### Reward Points System
Users earn points for every successful e-waste pickup or drop-off.
### Awareness & Education Section
A dedicated section that shares information, tips, and awareness content about e-waste and eco-friendly habits.
### Localized & Community-Focused
Designed with small communities in mind, making it practical for real-world adoption.
